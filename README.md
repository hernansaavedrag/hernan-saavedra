# Sitio personal - Hernán Saavedra González (GitHub Pages)

## Cómo publicarlo
1. Crea un repositorio en GitHub (por ejemplo: `hernan-saavedra`).
2. Sube los archivos:
   - `index.html`
   - `assets/style.css`
   - `assets/script.js`
3. Ve a: **Settings → Pages**
4. En **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: `main` / `root`
5. Guarda. Tu sitio quedará en:
   `https://TU-USUARIO.github.io/TU-REPO/`

## Personalización rápida
En `index.html` cambia:
- `TU-USUARIO` y `TU-REPO` en canonical/og:url
- `TU-LINKEDIN`
- `TU.CORREO@DOMINIO.COM`

## Recomendado (SEO)
- Mantener nombre/cargo/descripcion consistentes con LinkedIn y otros perfiles.
